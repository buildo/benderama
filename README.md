## description

Instance-based database table generator.

example table:

| A | B | C |
| --- | --- | --- |
| 1 | 3.5 | hi |
| 2 | 2.5 | lol |
| 3 | 3.0 | boh |

generated row:

| A | B | C |
| --- | --- | --- |
| 4 | 3.1 | xasdqo123jk |

A is incremental - max +1 is taken, **B is sampled from the distribution of the data**, C is random :)
